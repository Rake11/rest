Git and Post in method attribute

The GET method in forms appends form data to the URL as query parameters, making it visible. It's ideal for retrieving data, like search queries, but not suitable for sensitive information due to its visibility.

POST---Sends form data in the request body, making it secure and suitable for submitting sensitive or large data (e.g., login forms).

why we use meta name="views"

The meta name="viewport"> tag ensures a webpage scales correctly on different devices, making it responsive. It controls the layout and zoom behavior to enhance user experience on various screen sizes.

Charset attribute in meta tag, property attribute, http attribute.

Charset Attribute in meta Tag--- The charset attribute specifies the character encoding for the HTML document, ensuring correct text rendering.

Property Attribute--- The property attribute is used in meta tags to define metadata for the document, often for social media or SEO purposes.

HTTP-Equiv Attribute--- The http-equiv attribute provides HTTP headers in meta tags, mimicking server-side responses for client-side configuration.
